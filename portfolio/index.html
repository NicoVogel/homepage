Healthcare

Patient Monitoring UI for Philips

Client: Philips
Status: Ongoing

Technologies Used

TypeScript, LitElements, SCSS, Eslint, Prettier, Playwright, Github Actions, VS Code Dev Containers, Nrwl Nx

Problem

Philips needed a scalable, future-proof frontend architecture for its next-generation patient monitoring system, designed to improve healthcare monitoring processes.

Solution

Led the transition to a mono-repo approach using NX tooling, improving the integration of multiple teams and enhancing the development workflow. Created an optimized dev environment to streamline onboarding and improve developer experience.

Outcome

Enabled faster development cycles, better collaboration across teams, and a scalable foundation for future expansions.
